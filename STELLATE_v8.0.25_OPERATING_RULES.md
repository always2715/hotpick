# STELLATE v8.0.25 운영 규칙

## 1. TOP 선정

TOP30은 콘텐츠 작성 가능 여부가 아니라 v8.0.20 관심도 상대순위로 먼저 확정합니다. 확정된 30개는 31위 이하 후보로 교체하지 않습니다.

## 2. 키워드

키워드는 v8.0.22 규칙에 따라 대표 대상 하나로 저장합니다.

- `가수 박서진 전국투어` → `박서진`
- `삼성전자 갤럭시 Z 폴드 공개` → `갤럭시 Z 폴드`
- `넷플릭스 오징어 게임 시즌 3` → `오징어 게임 시즌 3`

## 3. 콘텐츠 작성

선정된 키워드는 모두 조사·작성 대상입니다.

### 현재 이슈가 확인되는 경우

- 키워드 기본정보
- 현재 이슈
- STELLATE 인사이트
- 연관 뉴스·관련 영상·출처

### 현재 이슈가 명확하지 않은 경우

- 키워드가 무엇인지 확인
- 인물·제품·작품·기업·지역 등의 기본정보
- 확인된 주요 활동·특징
- STELLATE 인사이트
- 실제 확인 출처

공통 관심 증가 고정문은 사용하지 않습니다.

## 4. 상태

TOP30 작업 상태는 다음만 사용합니다.

- 대기
- 처리
- 생성
- 추가 검색·재작성
- 기술 실패
- 중단

TOP 작업에서는 `검토`, `pending`, `review`, `검토기준 미충족`을 사용하지 않습니다.

## 5. 반복 방지 및 중단

- 키워드당 최대 3회 처리
- 첫 시도 실패 시 검색어 확장
- 최대 시도 후 `KEYWORD_CONTENT_EXHAUSTED` 또는 실제 기술 오류로 종료
- 전체 실행 최대 18단계
- 전체 실행 최대 60분
- 관리자 `작업 중단` 버튼 제공
- 중단 요청 후 처리 중인 소작업이 종료되면 다음 배치·최종 공개 중단
- 중단·자동 제한 시 기존 공개 TOP 유지

## 6. 공개

30개 상세·피드·제목이 준비된 경우에만 원자적으로 공개합니다. 일부 결과는 공개하지 않습니다.
