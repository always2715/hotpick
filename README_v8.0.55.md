# STELLATE v8.0.55

v8.0.55는 TOP 후보의 2·3차 재시도에서 이전 조사 Fact와 출처를 버리지 않고 누적해, 충분한 근거가 여러 시도에 나뉘어 확보됐는데도 피드 본문 1,000자 미달로 반복 실패하던 문제를 수정합니다.

핵심 변경:
- 재시도별 Fact Ledger·출처·연관 자료 누적 및 URL·사실 중복 제거
- ETF·eSIM 같은 광범위 키워드는 발견 기사 제목의 사건 단서를 검색 힌트로만 사용
- `급등해`, `최소 164명` 같은 서술어·수량 조각을 TOP25 고정 전에 제외
- 충분한 서로 다른 Fact가 있으면 내부 안내문이나 반복 문장 없이 검증형 fallback을 1,000자 이상 구성
- 직전 정상 TOP의 공개본은 과거 분량 기준으로 작성됐더라도 혼합 공개 보충 후보로 유지
- TOP25 상대순위, 성공 20개 조기 종료, 신규 15~19개 혼합 공개, 누적 피드, 썸네일 500개 정책 유지

GitHub Desktop 적용에는 `STELLATE_v8.0.55_REPLACE_ONLY.zip`을 저장소 최상위에 덮어쓴 뒤 `APPLY_STELLATE_v8.0.55_CLEANUP.bat`을 실행하십시오.
