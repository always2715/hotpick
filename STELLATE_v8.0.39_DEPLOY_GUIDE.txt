3. 피드 0건 복구
======================================================================

1) 배포 후 공개 /feed 페이지를 새로고침
   - 인덱스가 비거나 오래됐으면 누적 공개 콘텐츠에서 자동 재구성
   - 콘텐츠 인덱스도 비어 있으면 현재 공개 TOP 스냅샷으로 비상 복구
2) 즉시 강제 복구가 필요하면 관리자 로그인
3) 관리자에서 `현재 TOP 피드 목록 재구성` 실행
4) 공개 /feed 페이지와 /api/feed 응답 건수 확인
5) 기존 공개 콘텐츠가 숨김·비공개 상태가 아닌데도 누락되는지 표본 확인

주의:
- Redis FLUSHALL 금지
- reset-feed로 기존 피드를 먼저 지우지 않음
- 새 TOP 작업을 시작하기 전에 기존 피드 삭제 금지
