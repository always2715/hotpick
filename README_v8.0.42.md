# STELLATE v8.0.42

기준 버전: v8.0.41
작성일: 2026-06-24

## 핵심 수정

1. 피드 인덱스, Hash, ZSET, 콘텐츠 인덱스가 비어 있어도 현재 공개 TOP의 상세 콘텐츠 원본을 직접 읽어 `/feed` 목록을 구성합니다.
2. Unsplash 100개 사전 풀을 TOP 작업 시작 전에 자동 점검하고, 비어 있으면 구축을 시도합니다.
3. Unsplash API 제한 등으로 100개를 한 번에 채우지 못해도 확보된 유효 이미지를 부분 풀로 즉시 사용하고 다음 실행에서 부족분을 보충합니다.
4. 현재 TOP에서 이미지가 없는 항목은 풀 구축·점검 후 자동으로 backfill 합니다. 기존 Unsplash 이미지와 관리자 수동 이미지는 유지합니다.
5. 이미지 풀이 없을 때 표시되던 불꽃 아이콘을 중립적인 STELLATE 카테고리 fallback으로 교체했습니다.

## 변경하지 않은 정책

- 관심도 상대순위 상위 25개 생성 후보 고정
- 성공 후보를 원래 selectionRank 순으로 정렬
- 상위 20개 공개 및 21~25위 성공 후보 승격
- 성공 후보 20개 미만이면 기존 TOP 유지
- 이미지 적합도는 TOP 점수·후보 성공 여부에 사용하지 않음

GitHub Desktop 적용 시 `STELLATE_v8.0.42_REPLACE_ONLY.zip`을 저장소 최상위에 덮어쓰고 전체 변경을 커밋·Push합니다.
