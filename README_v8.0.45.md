# STELLATE v8.0.45

기준 버전: v8.0.44  
작성일: 2026-06-25

## 핵심 변경

### 피드 상세 분량 정책

- 공개 가능한 최소 본문은 공백 포함 1,000자입니다.
- 확인된 Fact Ledger와 출처가 충분한 경우 약 5,000자를 권장합니다.
- 일반적인 권장 범위는 약 3,500~6,000자입니다.
- 1,000자 이상이면 5,000자에 미달해도 분량만으로 실패시키지 않습니다.
- 6,000자를 넘더라도 사실 반복·불필요한 일반론이 없다면 분량만으로 실패시키지 않습니다.
- 자료가 적을 때 같은 사실을 표현만 바꿔 반복하거나 전망·평가·추측을 추가해 5,000자를 맞추는 행위를 금지합니다.
- 확인 사실이 1,000자를 안정적으로 구성하기에도 부족하면 가짜 문장으로 채우지 않고 해당 후보를 실패 처리합니다.

### TOP 클릭 요약

- `/{slug}` 요약정보는 summary, why, points 합계 최대 1,000자를 유지합니다.
- 상세 본문은 `/feed/{slug}`에서 제공합니다.

## 유지 정책

- 관심도 상대순위 상위 25개 생성 후보 고정
- 성공 후보 상위 20개 공개
- 실패 후보 발생 시 21~25위 성공 후보 승격
- 성공 후보 20개 미만이면 기존 공개 TOP 유지
- 누적 피드 보존 및 과거 `content:*` 원본 복구
- Unsplash 500개 사전 풀
- 썸네일 적합도는 TOP 순위 점수와 완전 분리
