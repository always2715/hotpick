STELLATE v8.0.32 GitHub Desktop 교체파일
================================================

적용 대상: 기존 STELLATE v8.0.31 hotpick 저장소

1. GitHub Desktop에서 hotpick 저장소 선택
2. Repository > Show in Explorer
3. 이 폴더 안의 파일과 폴더를 모두 선택
4. 기존 hotpick 최상위 폴더에 붙여넣기
5. Windows 확인창에서 대상 폴더의 파일 덮어쓰기 선택
6. GitHub Desktop Changes 확인
7. Commit to main
8. Push origin
9. Vercel 배포 완료 후 /api/version 확인

정상 버전:
- appVersion 8.0.32
- contentVersion 128
- trendCacheVersion 48
- publicTopCount 20

중요:
- .git 폴더를 삭제하거나 교체하지 마세요.
- node_modules, .next, .env.local, .vercel은 복사하거나 커밋하지 마세요.
- 기존 실패 실행은 재개하지 말고 관리자에서 즉시 중단한 뒤 새 TOP20 작업을 시작하세요.
- 이번 교체본은 v8.0.31 기준입니다. 더 오래된 버전에는 전체본을 적용하세요.

권장 확인:
npm install
npm run test:v8032
npm run build
