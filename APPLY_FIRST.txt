================================================

이 폴더 안의 파일과 폴더만 기존 hotpick 저장소 최상위에 복사해 덮어쓰면 됩니다.

적용 위치 예:
C:\GitHub\hotpick

정상 위치:
C:\GitHub\hotpick\package.json
C:\GitHub\hotpick\lib\topConfig.js
C:\GitHub\hotpick\pages\admin.js

적용 순서:
1. GitHub Desktop에서 hotpick 저장소 선택
2. Repository > Show in Explorer
3. 이 교체폴더 안의 항목을 모두 선택
4. 기존 hotpick 폴더에 붙여넣기
5. 파일 교체 확인창에서 '대상 폴더의 파일 덮어쓰기' 선택
6. GitHub Desktop Changes에서 변경 파일 확인
7. node_modules, .next, .env.local, .vercel이 Changes에 없는지 확인
8. Summary: release: STELLATE v8.0.31 TOP20
9. Commit to main
10. Push origin
11. Vercel 자동 배포 확인
