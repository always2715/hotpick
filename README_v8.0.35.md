# STELLATE v8.0.35

STELLATE TOP20 안정화 흐름은 유지하면서 피드·상세 콘텐츠의 정확성, 읽기 경험, 출처 노출을 고도화한 버전입니다.

## 핵심 변경

- 공개 화면 하단에 `Powered by Upstash Redis · QStash` 소형 인프라 크레딧 추가
- 상세 상단에 핵심 요약, 배경·일정 맥락, 주요 포인트 3~5개 표시
- 상세에 검증 출처·공식 자료·확인 사실·관련 뉴스·영상 개수 표시
- 이미 생성된 Q&A를 실제 상세 화면에 표시
- 3~5개 역할형 섹션으로 본문 구성: 기본정보, 현재 사건, 맥락·수치·일정, 이용자 확인사항, 근거형 인사이트
- 동일 Fact를 여러 구역에서 문장만 바꿔 반복하는 구조 차단
- 결정론적 정확성 fallback도 자연스러운 존댓말, 조사, 일정·수치형 소제목으로 개선
- 피드 목록은 `why` 한 줄 대신 핵심 요약 2줄과 출처·사실 개수 표시
- 관련영상 URL만 있는 데이터도 표시 가능
- 연관 뉴스와 자료 출처 URL을 정규화해 중복은 한 번만 표시
- A/B/C 등급 목표 분량을 Fact 수에 맞게 현실화해 분량 채우기식 반복 문장 감소
