# STELLATE v8.0.25 릴리스 노트

v8.0.25는 TOP30 키워드가 확정된 뒤 콘텐츠 검토 기준으로 다시 탈락하던 문제를 수정합니다.

## 처리 원칙

```text
TOP 후보 수집
→ v8.0.20 관심도 기준으로 대표 키워드 30개 확정
→ 각 키워드의 정체·기본정보·현재 이슈 독립 조사
→ 현재 사건이 있으면 이슈형 피드
→ 현재 사건이 없으면 확인된 정체·활동·특징 기반 설명형 피드
→ 제목·요약정보·인스타카드 생성
→ 30건 준비 후 원자적 공개
```

`검토기준 미충족`은 TOP 키워드 탈락 사유로 사용하지 않습니다. 검색·저장·AI 호출과 같은 기술 오류만 별도로 표시합니다.

## 반복 방지

- 키워드별 최대 시도: 기본 3회
- 실행 최대 단계: 기본 18회
- 실행 최대 시간: 기본 60분
- 관리자 중단: 현재 작업 중단 버튼
- 중단 시: 다음 배치와 최종 공개 차단, 기존 공개 TOP 유지

환경변수로 허용 범위 안에서 조정할 수 있습니다.

- `TOP_KEYWORD_MAX_ATTEMPTS`
- `TOP_REFRESH_MAX_STEPS`
- `TOP_REFRESH_MAX_MINUTES`
