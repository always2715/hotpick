# STELLATE v8.0.45

이 버전은 v8.0.44의 누적 피드 보존 구조를 유지하면서 피드 상세 분량 기준을 현실적으로 조정합니다.

- 피드 상세 본문: 최소 1,000자
- 권장 목표: 확인 사실과 출처가 충분하면 약 5,000자
- 권장 범위: 약 3,500~6,000자
- 사실이 부족하면 5,000자를 맞추기 위해 반복·일반론·추측을 추가하지 않음
- 5,000자 미달 또는 6,000자 초과만으로 공개 실패 처리하지 않음
- TOP 클릭 요약정보: 최대 1,000자 유지

기존 TOP25 생성 후보 → 성공 TOP20 공개, 누적 피드, 예비 후보 승격, Unsplash 500개 썸네일 풀 정책은 유지합니다.

GitHub Desktop 적용에는 `STELLATE_v8.0.45_REPLACE_ONLY.zip`을 사용하고 전체를 저장소 최상위에 덮어쓴 뒤 `APPLY_STELLATE_v8.0.45_CLEANUP.bat`을 실행하십시오.
