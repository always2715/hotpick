# STELLATE v8.0.47

STELLATE 최신 전체 소스입니다.

이번 버전은 공개 UI 문구를 정리했습니다. 상단 메뉴는 `TOP20`, 메인 제목은 `지금 확인된 주요 이슈` / `실시간 검색 순위 - TOP20` 두 줄로 표시하며 오른쪽에 `3시간 단위 갱신` 안내를 추가했습니다.

TOP25 생성 후보에서 성공 TOP20을 공개하는 집계 방식, 누적 피드, 콘텐츠 분량 정책, NEW·유지·순위변동, Unsplash 500개 썸네일 풀은 변경하지 않았습니다.

GitHub Desktop 적용에는 `STELLATE_v8.0.47_REPLACE_ONLY.zip`을 사용하고 전체를 저장소 최상위에 덮어쓴 뒤 `APPLY_STELLATE_v8.0.47_CLEANUP.bat`을 실행하십시오.
