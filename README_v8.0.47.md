## 변경 사항

- 상단 메뉴의 `검색순위` 명칭을 `TOP20`으로 변경했습니다.
- 메인 상단 제목을 다음 두 줄로 변경했습니다.
  - 지금 확인된 주요 이슈

- 제목 오른쪽에 `3시간 단위 갱신`을 작게 표시합니다.
- 실제 최근 갱신 시각은 데스크톱에서 함께 표시하고 모바일에서는 주기 문구만 간결하게 표시합니다.
- TOP25 후보 선정, 성공 TOP20 공개, 누적 피드, 콘텐츠 생성, 순위변동, 썸네일 500개 정책은 변경하지 않았습니다.
