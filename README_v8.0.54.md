# STELLATE v8.0.54

v8.0.54는 TOP25를 필수 처리 수량이 아니라 최대 예비 후보 풀로 사용합니다.

## 핵심 변경

- 관심도 상대순위 상위 25개 후보 풀은 그대로 고정합니다.
- 1위부터 순서대로 조사·생성합니다.
- 정상 콘텐츠가 20개 확보되면 낮은 순위의 남은 후보를 더 생성하지 않습니다.
- 이미 처리된 더 높은 순위 `retry_wait` 후보는 최종 TOP20 순서에 영향을 줄 수 있으므로 제한 재시도를 마칩니다.
- 불필요한 후보는 실패가 아니라 `SKIPPED_NOT_NEEDED`로 기록합니다.
- 신규 정상 콘텐츠가 15~19개인 경우 직전 정상 TOP으로 부족분을 보충하는 v8.0.51 혼합 공개 정책은 유지합니다.
- TOP 관심도 점수, 후보 상대순위, 누적 피드, 순위변동, 썸네일 500개 정책은 변경하지 않습니다.

## 처리 예

- 1~20위 전부 성공: 20개 처리 후 21~25위 생략
- 2위·7위 실패, 21위·22위 성공: 정상 20개가 확보된 시점에 종료
- 정상 19개로 25위까지 처리: 신규 19개 + 직전 TOP 1개 혼합 공개
