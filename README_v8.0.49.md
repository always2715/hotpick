# STELLATE v8.0.49

## 핵심 수정

- 요약정보·피드 본문·포인트·Q&A에 노출되던 출처명, 도메인, 검증 절차, 동명이인 구분 안내, 원문 확인 안내 등 내부 편집 문구를 제거했습니다.
- 결정론적 Fact fallback이 분량을 채우기 위해 내부 안내문을 반복하던 구조를 제거했습니다.
- 새 콘텐츠에 내부 편집 문구가 포함되면 공개 검증에서 차단합니다.
- 기존 Redis 콘텐츠도 재생성하지 않고 공개 조회 시 정제합니다.
- `&nbsp;`, 숫자형 HTML entity, 비표준 공백을 정상 공백으로 변환합니다.
- `머니투데이  머니투데이. 머니투데이`처럼 반복된 출처 표시는 한 번으로 축약합니다.
- 피드 인덱스 스키마를 4로 올려 기존 인덱스가 새로운 정제 정책으로 다시 작성되도록 했습니다.
